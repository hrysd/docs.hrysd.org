このサイトに `theme-color` を設定した。

普段 [Nexus5X](http://docs.hrysd.org/2015/10/22/hello-nexus-5x/) を使用しているので、せめて自分の端末でくらいは綺麗に見たかった。

![theme-color](/assets/images/theme-color.png)

## 方法

以下の様なメタタグを追加するだけ。

```html
<meta name='theme-color' content='#a3e9a4'>
```
